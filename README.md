# Scale Model Color Manager — Demo

Open `index.html` in your browser to run the client-only demo.

Features implemented:
- List view for a single brand (selectable)
- Color swatch, code, name, brand label
- Quick in-stack toggle saved to `localStorage`

Files of interest:
- `index.html` — main page
- `script.js` — client logic
- `data/*.json` — seed data files
